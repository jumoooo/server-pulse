# Agent Critic Skills

## Language Separation
**Internal**: English. **User-facing**: Korean.

## Overview

Core functions: evaluate agent routing, validate plans, review code changes, config/rules health check.

## Skills

### 1. Evaluate Routing
Assess if correct agents were selected for the task.

### 2. Validate Plans
Check plan completeness, feasibility, risks.

### 3. Review Code Changes
Stability, efficiency, code quality from diff.

### 4. Config Health Check
When given .cursor/rules or agents, find conflicts, duplicates, suggest improvements.

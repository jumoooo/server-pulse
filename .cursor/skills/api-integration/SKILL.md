# API Integration Skills

## Language Separation
**Internal**: English. **User-facing**: Korean.

## Overview

Core functions: API client creation, model generation, service layer with error handling, type safety.

## Skills

### 1. Analyze API Requirements
Endpoint structure, authentication, request/response formats.

### 2. Create API Client
HTTP client setup (dio, http), base URL, headers, interceptors.

### 3. Generate Models
JSON serialization, fromJson/toJson, type safety.

### 4. Implement Service Layer
Error handling, retry logic, response parsing.

### 5. Integrate with State Management
Provider/Riverpod integration for API state.

## ✅ 완료된 작업

### 1. 매직 넘버 상수화 ✅

**생성된 파일**: `lib/constants/app_constants.dart`

**주요 상수 정의**:
- **EdgeInsets 상수**: 다이얼로그 패딩, 기본 패딩, 작은 패딩, 헤더/컨텐츠 패딩, 간격 등
- **BorderRadius 상수**: 다이얼로그, 버튼, 칩/배지, 카드 모서리 반경
- **Size 상수**: 아이콘 크기, 버튼 최소 크기, 날짜 셀 크기
- **FontSize 상수**: 매우 작은, 작은, 기본 폰트 크기
- **FontWeight 상수**: 보통, 세미볼드, 볼드
- **BorderWidth 상수**: 얇은, 기본, 중간, 두꺼운, 강조 테두리 두께
- **Opacity/Alpha 상수**: 투명도 값 (20, 24, 26, 36, 160, 180)
- **Height/Width 비율 상수**: 다이얼로그 높이 비율, 화면 분할 비율
- **Duration 상수**: 애니메이션 지속 시간 (300ms)

**편의 메서드**:
- `dialogInsetPadding`, `dialogInsetPaddingSmall`
- `defaultPadding`, `headerPadding`, `contentPadding`
- `smallPadding`, `tinyPadding`, `microPadding`, `buttonPadding`
- `dialogBorderRadius`, `buttonBorderRadius`, `chipBorderRadius`, `cardBorderRadius`
- `buttonMinSizeBox`
- `borderSide()`, `borderSideMedium()`, `borderSideThick()`, `borderSideEmphasis()`

### 2. 앱 설정 상수 정의 ✅

**생성된 파일**: `lib/constants/app_config.dart`

**주요 설정 정의**:
- **데이터 관리 설정**:
  - `maxUndoStackSize = 5` (삭제 취소 스택 최대 크기)
  - `maxLunarCacheSize = 500` (음력 캐시 최대 크기)
  - `maxDateFormatCacheSize = 100` (날짜 포맷 캐시 최대 크기)
- **할일 관리 설정**:
  - `defaultMaxTodoCount = null` (기본 할일 개수 제한)
  - `todoCountWarningThreshold = null` (할일 개수 경고 임계값)
- **UI 설정**:
  - `animationDurationMs = 300` (애니메이션 지속 시간)
  - `skeletonAnimationDurationMs = 250` (스켈레톤 애니메이션 지속 시간)
- **검색/통계/백업 설정**: 향후 확장을 위한 설정 값들

## 🔄 롤백 방법

### 1. 새로 생성된 파일 삭제

```bash
# 상수 파일 삭제
rm lib/constants/app_constants.dart
rm lib/constants/app_config.dart
```

### 2. 수정된 파일 복원

Git을 사용하여 롤백:

```bash
# 특정 파일만 롤백
git checkout HEAD -- lib/providers/todo_provider.dart
git checkout HEAD -- lib/utils/date_utils.dart
git checkout HEAD -- lib/widgets/
git checkout HEAD -- lib/screens/calendar_screen.dart

# 또는 모든 변경사항 롤백
git checkout HEAD -- lib/
```

### 3. 수동 롤백 (Git 사용 불가 시)

각 파일에서 다음 변경사항을 되돌려야 합니다:

**lib/providers/todo_provider.dart**:
- `import 'package:flutter_calender/constants/app_config.dart';` 제거
- `AppConfig.maxUndoStackSize` → `5`로 변경

**lib/utils/date_utils.dart**:
- `import 'package:flutter_calender/constants/app_config.dart';` 제거
- `AppConfig.maxLunarCacheSize` → `500`으로 변경

**나머지 위젯 파일들**:
- `import 'package:flutter_calender/constants/app_constants.dart';` 제거
- 모든 `AppConstants.*` 사용을 원래 하드코딩된 값으로 복원

자세한 내용은 `rollback-guide.md`를 참조하세요.
